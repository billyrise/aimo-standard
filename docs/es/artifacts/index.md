# Artefactos

Esta sección es un centro para artefactos legibles por máquina y legibles por humanos que implementan el AIMO Standard.

- **Paquete de Evidencia**: estructura, índice, trazabilidad — [Paquete de Evidencia](evidence-bundle.md).
- **Requisitos Mínimos de Evidencia**: lista de verificación de nivel DEBE por ciclo de vida — [Requisitos Mínimos de Evidencia](minimum-evidence.md).
- **Taxonomía / Códigos / Diccionario**: [Taxonomía](../standard/current/03-taxonomy.md), [Códigos](../standard/current/04-codes.md), [Diccionario](../standard/current/05-dictionary.md).
- **Plantillas de evidencia**: [Plantilla EV](../standard/current/06-ev-template.md) y esquemas relacionados.
- **Esquemas**: JSON Schema bajo `schemas/jsonschema/` en el repositorio.
- **Ejemplos**: muestras mínimas bajo [Ejemplos](../examples/index.md).

Para empaquetado listo para auditoría, consulte el [Paquete de Confianza](../governance/trust-package.md).
