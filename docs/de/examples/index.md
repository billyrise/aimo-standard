# Beispiele

Dieser Abschnitt verweist auf End-to-End- und minimale Beispiele, die zeigen, wie Artefakte zusammengestellt werden.

- **Minimales Bundle**: `examples/minimal/` im Repository (z.B. `sample_ev.json`, `sample_dictionary.json`).
- **Evidence Bundle (minimal)**: `examples/evidence_bundle_minimal/` — root.json mit optionalem request, review, change_log; siehe [Evidence Bundle](../artifacts/evidence-bundle/) und [Mindestanforderungen](../artifacts/minimum-evidence/).
- **Schema-Ausrichtung**: Validieren gegen `schemas/jsonschema/` und [Standard](../standard/current/).
- **Prüfungsfertige Pakete**: siehe das [Trust Package](../governance/trust-package/).

Detaillierte Szenario-Abdeckung wird in späteren Änderungseinheiten hinzugefügt.
