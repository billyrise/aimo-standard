# 商標

本ページでは、AIMO 関連の名称の商標上の取扱いと使用ガイドラインを説明する。

**English notice (key facts):** Only AIMOaaS has been filed for trademark registration by RISEby Inc. (pending; registration not yet granted). "AIMO" is a registered trademark owned by third parties in various jurisdictions (including Japan); RISEby Inc. does not claim ownership. "AIMO Standard" and "Certified AIMO" are not registered by RISEby. "Certified AIMO" is a provisional name; no active certification scheme exists.

正確な商標の帰属・出願状況は上記の English notice を参照してください。RISEby が商標出願しているのは AIMOaaS のみです。

## 表記ガイダンス

商標出願中の期間は以下のルールが適用される：

- **許容**：「商標（出願中）」、「application pending」、TM 記号（任意）
- **禁止**：「登録商標」、® 記号、登録番号

詳細は完全なポリシーを参照。

## 許可される使用

### 説明的な参照

互換性を説明するために AIMO 商標を使用できる：

- 「AIMO Standard と互換」
- 「AIMO Standard v0.0.1 を実装」
- 「AIMO Standard 形式でエビデンスをエクスポート」

**条件**：

1. 仕様のすべての MUST 要件を満たす
2. 公式バリデータをルール変更なしで通過
3. 特定のバージョンを明記

### 学術利用

以下で AIMO Standard を引用できる：

- 学術論文
- 研究出版物
- 研修資料（帰属表示付き）

引用形式は [引用方法](../cite/) を参照。

## 禁止される使用

| 禁止事項 | 理由 |
| -------- | ---- |
| 「AIMO」を製品/会社名として使用 | 混乱を招く |
| 認証なしに「Certified AIMO」を主張 | 認証プログラムが存在しない |
| ドメイン名に商標を使用 | 公式リソースと競合 |
| 公式な推奨を暗示 | ユーザーを誤解させる |

## 適合性の主張

### 主張できること

> 「本製品は AIMO Standard v0.0.1 を実装しています。」

### 主張できないこと

- 「AIMO 認証」（認証プログラムなし）
- 「公式 AIMO 実装」（メンテナでない限り）
- 「AIMO Standard 推奨」
