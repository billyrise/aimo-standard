# Artifacts

本セクションは、AIMO Standard を実装する機械可読・人間可読アーティファクトのハブです。

- **Evidence Bundle**: 構造、TOC、追跡可能性 — [Evidence Bundle](evidence-bundle/)。
- **Evidence Bundle Coverage Map（テンプレ）**: 監査向け一枚まとめ（スコープ、エビデンス種別、コントロール対応、除外、整合性）— [Evidence Bundle Coverage Map](evidence-bundle-coverage-map/)（informative）。
- **Minimum Evidence Requirements**: ライフサイクル別 MUST チェックリスト — [Minimum Evidence Requirements](minimum-evidence/)。
- **タクソノミー／コード／辞書**: [Taxonomy](../../standard/current/03-taxonomy/)、[Codes](../../standard/current/04-codes/)、[Dictionary](../../standard/current/05-dictionary/)。
- **証跡テンプレート**: [EV Template](../../standard/current/06-ev-template/) および関連スキーマ。
- **スキーマ**: リポジトリの `schemas/jsonschema/`。
- **例**: 最小サンプルは [Examples](../../examples/) を参照。

監査対応のパッケージ化は [Trust Package](../../governance/trust-package/) を参照。
