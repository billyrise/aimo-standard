# 상표

이 페이지는 본 사이트에서 참조하는 명칭의 상표 상태 및 사용 지침을 설명합니다.

**English notice (key facts):** Only AIMOaaS has been filed for trademark registration by RISEby Inc. (pending; registration not yet granted). "AIMO" is a registered trademark owned by third parties in various jurisdictions (including Japan); RISEby Inc. does not claim ownership. "AIMO Standard" and "Certified AIMO" are not registered by RISEby. "Certified AIMO" is a provisional name; no active certification scheme exists.

정확한 상표 귀속 및 출원 상태는 위 English notice를 참조하세요.

## 마킹 지침

상표 출원이 진행 중인 동안:

- **허용**: "상표 출원 중", "TM" 기호 (선택)
- **금지**: "등록 상표", ® 기호, 등록 번호

자세한 내용은 전체 정책을 참조하세요.

## 허용된 사용

### 설명적 참조

호환성을 설명하기 위해 AIMO 마크를 사용할 수 있습니다:

- "AIMO Standard와 호환"
- "AIMO Standard v0.0.1 구현"
- "AIMO Standard 형식으로 증거 내보내기"

**요구사항**:

1. 사양의 모든 MUST 요구사항 충족
2. 규칙 수정 없이 공식 검증기 통과
3. 특정 버전 명시

### 학술적 사용

다음에서 AIMO 표준을 인용할 수 있습니다:

- 학술 논문
- 연구 출판물
- 교육 자료 (저작자 표시 포함)

인용 형식은 [인용 방법](../cite/)을 참조하세요.

## 금지된 사용

| 금지 | 이유 |
| ---------- | ------ |
| 제품/회사 이름에 "AIMO" 사용 | 혼란 초래 |
| 인증 없이 "Certified AIMO" 주장 | 인증 프로그램 없음 |
| 도메인 이름에 마크 사용 | 공식 리소스와 충돌 |
| 공식 승인 암시 | 사용자 오도 |

## 적합성 주장

### 주장할 수 있는 것

> "이 제품은 AIMO Standard v0.0.1을 구현합니다."

### 주장할 수 없는 것

- "AIMO Certified" (인증 프로그램 없음)
- "Official AIMO implementation" (메인테이너가 아닌 경우)
- "AIMO Standard 승인"
