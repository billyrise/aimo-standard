# 산출물

이 섹션은 AIMO 표준을 구현하는 기계 판독 가능 및 사람 판독 가능 산출물의 허브입니다.

- **증거 번들**: 구조, 목차, 추적성 — [증거 번들](evidence-bundle.md).
- **최소 증거 요구사항**: 생명주기별 MUST 수준 체크리스트 — [최소 증거 요구사항](minimum-evidence.md).
- **분류체계 / 코드 / 딕셔너리**: [분류체계](../standard/current/03-taxonomy.md), [코드](../standard/current/04-codes.md), [딕셔너리](../standard/current/05-dictionary.md).
- **증거 템플릿**: [EV 템플릿](../standard/current/06-ev-template.md) 및 관련 스키마.
- **스키마**: 저장소의 `schemas/jsonschema/` 아래 JSON Schema.
- **예제**: [예제](../examples/index.md) 아래의 최소 샘플.

감사 준비 패키징에 대해서는 [신뢰 패키지](../governance/trust-package.md)를 참조하세요.
