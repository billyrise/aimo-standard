# 개요

**AIMO**는 **AI Management Office**의 약칭입니다. AIMO 표준은 다음을 정의합니다:
- 공유 분류체계
- 코드 시스템
- 초기 딕셔너리
- EV 템플릿
- 검증기 검사 (사양 + 최소 참조 구현)

이 저장소는 다음을 게시합니다:
- 사람이 읽을 수 있는 사양 (HTML)
- 기계 판독 가능 산출물 (스키마/템플릿/예제)
- 공식 PDF 릴리스
