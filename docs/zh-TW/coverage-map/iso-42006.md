# ISO/IEC 42006 對應

> 可追溯性捷徑：[分類法](../../standard/current/03-taxonomy/) → [最低證據](../../artifacts/minimum-evidence/) → [驗證器](../../validator/) → [符合性](../../conformance/)。

本頁說明 **ISO/IEC 42006** 以及 AIMO Standard 如何協助採用者準備 AIMS 審計。內容僅供參考；不構成認證或法律意見。

## 什麼是 ISO/IEC 42006？

**ISO/IEC 42006** 規定依據 **ISO/IEC 42001** 對 AI 管理系統（AIMS）進行**審計與認證**之機構的要求。它以 ISO/IEC 17021-1（適合性評鑑 — 提供管理系統審計與認證之機構的要求）為基礎，於 **2025 年**（例如 2025-07）發布，明確規範執行 AIMS 審計之認證機構的能力與流程期望。

## 對 AIMO 採用者的重要性

適用 ISO/IEC 42006 的認證機構將要求管理系統控制之**可審計證據**：文件化方針、角色、營運控制、監控與改進。AIMO 協助將該證據結構化為一致的組合（manifest、雜湊、生命週期紀錄），並提供結構一致性用的驗證器。**並不**取代審計員的判斷或認證決定。

## AIMO 支援

| AIMO 元件 | 如何支援審計準備 |
| --- | --- |
| **證據包 + 驗證器** | 可審計結構：manifest、object_index、payload_index、hash_chain、signing。驗證器檢查存在與完整性；輸出可附於組合供審計員使用。 |
| **符合性級別** | Foundation / Operational / Audit-Ready 描述證據成熟度（可追溯性、生命週期軌跡、聲明、交接索引）。表示準備程度，非認證級別。 |
| **對應圖 + 設定檔** | 可解釋性與要求可追溯性：從 AIMO 人工產物到 ISO 42001 條款族乃至 EU 等框架的對應。 |

級別定義請見 [符合性](../../conformance/)；基盤標準對應請見 [ISO/IEC 42001](iso-42001.md)。
