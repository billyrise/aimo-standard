# 符合性

本節定義組織如何聲明符合 AIMO 標準以及預期的證據。

## 目的

- **您可以聲明的內容**：符合 AIMO 標準的證據結構和可追溯性要求。
- **您不能聲明的內容**：法律合規性、認證或監管核准。請參閱[責任邊界](../governance/responsibility-boundary.md)。

## 預期證據

為證明符合性，組織必須提供：

1. **證據包**：根據[人工產物](../artifacts/index.md)和[最低證據要求](../artifacts/minimum-evidence.md)的結構化 EV 記錄。
2. **驗證器結果**：驗證器的通過輸出 — 請參閱[驗證器](../validator/index.md)。
3. **版本參照**：明確的 AIMO 標準版本 — 請參閱[版本](../standard/versions/index.md)。

## 不過度聲明

!!! warning "重要"
    符合 AIMO 標準表示**證據就緒和可追溯性**，而非法律合規性或認證。請參閱[責任邊界](../governance/responsibility-boundary.md)。
