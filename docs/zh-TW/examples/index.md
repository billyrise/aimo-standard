# 範例

本節指向端對端和最小範例，展示人工產物如何組裝。

- **最小套件**：儲存庫中的 `examples/minimal/`（例如 `sample_ev.json`、`sample_dictionary.json`）。
- **證據包（最小）**：`examples/evidence_bundle_minimal/` — root.json 包含選用的 request、review、change_log；請參閱[證據包](../artifacts/evidence-bundle.md)和[最低證據](../artifacts/minimum-evidence.md)。
- **結構描述對齊**：對照 `schemas/jsonschema/` 和[標準](../standard/current/index.md)進行驗證。
- **稽核就緒打包**：請參閱[信任套件](../governance/trust-package.md)。

詳細的情境覆蓋範圍將在後續變更單元中新增。
