# Vue d'ensemble

Le standard AIMO définit :
- une taxonomie partagée
- un système de codes
- un dictionnaire initial
- un modèle EV
- des vérifications du validateur (spec + implémentation de référence minimale)

Ce dépôt publie :
- une spécification lisible par l'humain (HTML)
- des artefacts lisibles par machine (schémas/modèles/exemples)
- des versions PDF officielles
