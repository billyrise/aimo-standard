# Versioning & Reference Policy

This page is available in **English** and **Japanese**.

- [Versioning & Reference Policy (EN)](https://standard.aimoaas.com/en/standard/current/00-versioning-reference-policy.md)
- [バージョニング・参照方針 (JA)](https://standard.aimoaas.com/ja/standard/current/00-versioning-reference-policy.md)

Summary: `/X.Y.Z/` is immutable; `/latest/` points only to the latest stable release; `/dev/` is development and must not be used for audit citation.
