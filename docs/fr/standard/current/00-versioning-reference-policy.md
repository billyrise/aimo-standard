# Versioning & Reference Policy

This page is available in **English** and **Japanese**.

- [Versioning & Reference Policy (EN)](../../../en/standard/current/00-versioning-reference-policy/)
- [バージョニング・参照方針 (JA)](../../../ja/standard/current/00-versioning-reference-policy/)
