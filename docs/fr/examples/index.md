# Exemples

Cette section pointe vers des exemples de bout en bout et minimaux qui montrent comment les artefacts sont assemblés.

- **Lot minimal** : `examples/minimal/` dans le dépôt (ex. `sample_ev.json`, `sample_dictionary.json`).
- **Lot de preuves (minimal)** : `examples/evidence_bundle_minimal/` — root.json avec demande, revue, change_log optionnels ; voir [Lot de preuves](../artifacts/evidence-bundle/) et [Preuves minimales](../artifacts/minimum-evidence/).
- **Alignement des schémas** : validez par rapport à `schemas/jsonschema/` et [Standard](../standard/current/).
- **Packaging prêt pour l'audit** : voir le [Package de confiance](../governance/trust-package/).

La couverture détaillée des scénarios est ajoutée dans les unités de changement ultérieures.
