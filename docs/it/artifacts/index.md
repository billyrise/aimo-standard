Questa sezione è un hub per artefatti leggibili da macchina e da umani che implementano lo Standard AIMO.

- **Evidence Bundle**: struttura, TOC, tracciabilità — [Evidence Bundle](evidence-bundle.md).
- **Requisiti Minimi di Evidence**: checklist MUST-level per ciclo di vita — [Requisiti Minimi di Evidence](minimum-evidence.md).
- **Tassonomia / Codici / Dizionario**: [Tassonomia](../standard/current/03-taxonomy.md), [Codici](../standard/current/04-codes.md), [Dizionario](../standard/current/05-dictionary.md).
- **Template di evidence**: [Template EV](../standard/current/06-ev-template.md) e schemi correlati.
- **Schemi**: JSON Schema sotto `schemas/jsonschema/` nel repository.
- **Esempi**: campioni minimi sotto [Esempi](../examples/index.md).

Per il packaging pronto per l'audit, vedere il [Trust Package](../governance/trust-package.md).
