# Esempi

Questa sezione punta a esempi end-to-end e minimali che mostrano come vengono assemblati gli artefatti.

- **Bundle minimale**: `examples/minimal/` nel repository (es. `sample_ev.json`, `sample_dictionary.json`).
- **Evidence Bundle (minimale)**: `examples/evidence_bundle_minimal/` — root.json con opzionali request, review, change_log; vedere [Evidence Bundle](../artifacts/evidence-bundle/) e [Requisiti Minimi di Evidence](../artifacts/minimum-evidence/).
- **Allineamento allo schema**: validare contro `schemas/jsonschema/` e [Standard](../standard/current/).
- **Packaging pronto per l'audit**: vedere il [Trust Package](../governance/trust-package/).

La copertura dettagliata degli scenari viene aggiunta nelle unità di modifica successive.
