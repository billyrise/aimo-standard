# 商标

本页说明本网站所涉名称的商标状态与使用指引。

**English notice (key facts):** Only AIMOaaS has been filed for trademark registration by RISEby Inc. (pending; registration not yet granted). "AIMO" is a registered trademark owned by third parties in various jurisdictions (including Japan); RISEby Inc. does not claim ownership. "AIMO Standard" and "Certified AIMO" are not registered by RISEby. "Certified AIMO" is a provisional name; no active certification scheme exists.

准确的商标归属与申请状态请见以上 English notice。

## 标注指南

在商标申请期间：

- **允许**："trademark application pending"、"TM" 符号（可选）
- **禁止**："Registered trademark"、® 符号、注册号

请参阅完整政策了解详情。

## 允许的使用

### 描述性引用

您可以使用 AIMO 标识来描述兼容性：

- "Compatible with AIMO Standard"
- "Implements AIMO Standard v0.0.1"
- "Exports evidence in AIMO Standard format"

**要求**：

1. 满足规范中的所有 MUST 要求
2. 通过官方验证器而不修改规则
3. 声明特定版本

### 学术使用

您可以在以下内容中引用 AIMO 标准：

- 学术论文
- 研究出版物
- 培训材料（需归属）

请参阅 [如何引用](cite.md) 了解引用格式。

## 禁止的使用

| 禁止 | 原因 |
| ---------- | ------ |
| 使用 "AIMO" 作为您的产品/公司名称 | 造成混淆 |
| 未经认证声称 "Certified AIMO" | 不存在认证计划 |
| 在域名中使用标识 | 与官方资源冲突 |
| 暗示官方背书 | 误导用户 |

## 符合性声明

### 您可以声明的

> "This product implements AIMO Standard v0.0.1."

### 您不能声明的

- "AIMO Certified"（没有认证计划）
- "Official AIMO implementation"（除非是维护者）
- "Endorsed by AIMO Standard"
